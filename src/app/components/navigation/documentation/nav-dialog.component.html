About

Web application using Angular and router . The application has 2 pages:

  • showpage;
  • editor.

(each page in its own url).

Task Manager Showpage

  • A list of created elements with the date of their creation is displayed;
  • The list header has column names: name, date and time of creation, date and time of execution;
  • Each item has a drop-down menu that allows you to move the item up or down the list;
  • Clicking on an element opens a dialog with detailed field information.

Editor Page

On the editor page, you can:

  1. Add element:
    • Adding is done using a popup consisting of fields:
      • Name
      • Discription
      • Deadline
    • When creating an element, the Created date (Date + Time) field is created automatically.
  2. Copy element;
  3. Remove element.

Additional Features

  • The table can be sorted by clicking on the header field;
  • When the due date is reached, a toaster notification will appear with information about the item;
  • Validation and display of errors in the fields of the creation form;
